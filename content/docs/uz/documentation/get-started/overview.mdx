title: Umumiy Ko‘rinish

---

DVoice hujjatlariga xush kelibsiz! Ushbu qo‘llanma DVoice platformasining imkoniyatlarini o‘rganish uchun boshlang‘ich nuqta bo‘lib xizmat qiladi, u ilg‘or matnni nutqqa (TTS) va nutqni matnga (STT) aylantirish xizmatlarini taqdim etadi. Agar siz API’larimizni integratsiya qilmoqchi bo‘lgan dasturchi yoki takliflarimizni tushunmoqchi bo‘lgan foydalanuvchi bo‘lsangiz, bu hujjatlar sizga yordam beradi.

## Boshlanish

DVoice bilan sayohatingizni quyidagi bo‘limlarni o‘rganishdan boshlang:

- [Autentifikatsiya](./authentication) API’ga xavfsiz kirish uchun.
- [Modellar](./models) mavjud ovozlar va tillar haqida ma’lumot olish uchun.

<Callout type="info">
  API tokeningizni yaratish uchun [voice.airi.uz](https://voice.airi.uz) saytiga
  tashrif buyuring.
</Callout>

## Imkoniyatlar

Asosiy funksiyalarimizni o‘rganing:

- [Matnni nutqqa](../capabilities/text-to-speech) tabiiy ovozga aylantirish uchun.
- [Nutqni matnga](../capabilities/speech-to-text) audio’ni matnga aylantirish uchun.

## Dasturchilar uchun Qo‘llanma

Dasturchilar uchun quyidagilar mavjud:

- [SDK kutubxonalari](../developer-guide/sdk-libraries) DVoice’ni Node.js, Python va boshqalar bilan integratsiya qilish uchun.

## Manbalar

Qo‘shimcha ma’lumot quyidagi bo‘limlarda mavjud:

- [Sizning ma’lumotlaringiz](../resources/your-data) ma’lumotlarni qayta ishlash siyosati uchun.
- [Foydalanish chegaralari](../resources/rate-limits) foydalanish bo‘yicha ko‘rsatmalar uchun.

## Boshqa

Savollaringiz bormi? Umumiy savollar va javoblarni ko‘rish uchun [FAQ’imizga](../faq) tashrif buyuring.

Hujjatlar va API havolasi o‘rtasida o‘tish uchun tanlash tugmasidan foydalaning.
