# Stream TTS

Эндпоинт **Stream TTS** позволяет транслировать аудио по мере его генерации через WebSocket-соединение.

## Эндпоинт

`GET wss://oyqiz.airi.uz/api/v2/tts/stream`

## Аутентификация

Включите ваш API-токен в заголовке:

```
token: <your-token>
```

## Обязательные Поля

- **text**: Строка с текстом для преобразования в речь.
- **model**: Модель голоса (например, `"Iroda"`, `"Surayyo_v2"`).
- **format**: Формат аудио. Один из:
  - `MP3`
  - `WAV`
  - `AAC`
  - `OGG`
  - `FLAC`

## Дополнительные Поля

- **data**: Объект для специфичных для модели опций.
  - Для `Surayyo_v2` можно передать поле `emo` со значениями:
    - `Happy`
    - `Strict`
    - `Neural`

    **Примечания**:
    - Параметры (`text`, `model`, `format`) передаются как параметры запроса в URL.
    - Используйте `+` или `%20` для пробелов в параметре `text` (например, "Haq olinur, berilmas" становится "Привет+мир!").
    - Флаг `--binary` обеспечивает правильную обработку бинарных данных (аудио-фрагментов).
    - Вывод сохраняется в `output.mp3`. Убедитесь, что ваша система поддерживает воспроизведение MP3 или конвертацию.

  <Tab value="Node.js">
    ```javascript
    const WebSocket = require('ws');
    const fs = require('fs');

    // Замените '<your-token>' на ваш действительный API-токен
    const params = new URLSearchParams({
        text: 'Haq olinur, berilmas',
        model: 'Iroda',
        format: 'mp3'
    });
    const ws = new WebSocket(`ws://oyqiz.airi.uz/stream?${params.toString()}`, {
        headers: {
            'token': '<your-token>'
        }
    });

    ws.on('open', () => {
        console.log('WebSocket-соединение установлено');
        // Начальное сообщение не требуется; параметры в URL
    });

    const outputStream = fs.createWriteStream('output.mp3');
    ws.on('message', (data, isBinary) => {
        let buffer;
        if (isBinary) {
            // Обработка бинарных данных (например, Buffer, ArrayBuffer или массив Buffers)
            if (data instanceof Buffer) {
                buffer = data;
            } else if (data instanceof ArrayBuffer) {
                buffer = Buffer.from(new Uint8Array(data));
            } else if (Array.isArray(data)) {
                buffer = Buffer.concat(data);
            } else {
                console.warn('Неподдерживаемый формат бинарного сообщения');
                return;
            }
            console.log('Получен аудио-фрагмент:', buffer.length, 'байтов');
            outputStream.write(buffer);
        } else {
            // Обработка текстовых метаданных
            console.log('Получены метаданные:', data.toString());
        }
    });

    ws.on('close', () => {
        console.log('Трансляция завершена');
        outputStream.end();
    });

    ws.on('error', (error) => {
        console.error('Ошибка:', error.message);
        outputStream.end();
    });
    ```
    **Примечания**:
    - Установите библиотеку `ws`: `npm install ws`.
    - Параметры (`text`, `model`, `format`) передаются как параметры запроса в URL.
    - Аудио-фрагменты сохраняются в `output.mp3`.
    - Убедитесь, что токен действителен, чтобы избежать ошибок аутентификации.

  <Tab value="Python">
    ```python
    import websocket
    import urllib.parse

    # Замените '<your-token>' на ваш действительный API-токен
    params = {
        "text": "Haq olinur, berilmas",
        "model": "Iroda",
        "format": "mp3"
        # "indexes": true  # Необязательно: раскомментируйте, если нужно (не поддерживается этим API)
    }
    query_string = urllib.parse.urlencode(params)
    ws_url = f"ws://oyqiz.airi.uz/stream?{query_string}"
    headers = {
        "token": "<your-token>"
    }

    def on_open(ws):
        print("WebSocket-соединение установлено")
        # Начальное сообщение не требуется; параметры в URL

    def on_message(ws, message):
        if isinstance(message, bytes):
            print(f"Получен аудио-фрагмент: {len(message)} байтов")
            with open("output.mp3", "ab") as f:
                f.write(message)
        else:
            print(f"Получены метаданные: {message}")

    def on_error(ws, error):
        print("Ошибка:", error)

    def on_close(ws, close_status_code, close_msg):
        print("Трансляция завершена")

    ws = websocket.WebSocketApp(
        ws_url,
        header=headers,
        on_open=on_open,
        on_message=on_message,
        on_error=on_error,
        on_close=on_close
    )

    ws.run_forever()
    ```
    **Примечания**:
    - Установите библиотеку `websocket-client`: `pip install websocket-client`.
    - Параметры (`text`, `model`, `format`) передаются как параметры запроса в URL.
    - Аудио-фрагменты добавляются в `output.mp3` (используйте режим `"ab"` для бинарного добавления).
    - Убедитесь, что токен действителен, чтобы избежать ошибок аутентификации.
