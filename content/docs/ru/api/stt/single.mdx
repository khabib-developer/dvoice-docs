**Эндпоинт STT** позволяет преобразовывать аудио в точные текстовые транскрипции с помощью одного HTTP POST-запроса. Этот эндпоинт разработан для удобства использования с автоматическим определением языка при необходимости. Обновлено по состоянию на 19 августа 2025 года.

## Эндпоинт

`POST https://oyqiz.airi.uz/api/v2/stt`

## Аутентификация

Включите ваш API-токен в заголовке запроса:

```
token: <your-token>
```

## Тело Запроса

| Поле     | Тип    | Обязателен | Описание                                                                                         |
| -------- | ------ | ---------- | ------------------------------------------------------------------------------------------------ |
| audio    | файл   | ✅         | Аудиофайл для транскрипции (например, "audio.mp3").                                              |
| language | string | ❌         | Код языка ISO 639-1 (например, "uz" для узбекского); определяется автоматически, если не указан. |

**Примечания**:

- Используйте флаг `-F` для загрузки файла `audio` (например, `audio.mp3`).
- Замените `YOUR_API_TOKEN` на ваш действительный API-токен.
- Эндпоинт ограничен 5 минутами аудио в минуту на один API-токен.

```bash
curl -X POST "https://oyqiz.airi.uz/api/v2/stt" \
  -H "Content-Type: multipart/form-data" \
  -H "token: YOUR_API_TOKEN" \
  -F "audio=@audio.mp3" \
  -F "language=uz"
```

**Примечания**:

- Установите зависимости: `npm install node-fetch form-data`.
- Замените `YOUR_API_TOKEN` на ваш действительный токен.
- Убедитесь, что путь к файлу `audio.mp3` указан верно.

```javascript
import fetch from "node-fetch";
import fs from "fs";

async function stt() {
  const formData = new FormData();
  formData.append("audio", fs.createReadStream("audio.mp3"));
  formData.append("language", "uz");

  const response = await fetch("https://oyqiz.airi.uz/api/v2/stt", {
    method: "POST",
    headers: {
      token: "YOUR_API_TOKEN",
    },
    body: formData,
  });

  const data = await response.json();
  console.log("Транскрипция:", data.transcript);
}

stt();
```

**Примечания**:

- Установите библиотеку: `pip install requests`.
- Замените `YOUR_API_TOKEN` на ваш действительный токен.
- Проверьте доступность и читаемость файла `audio.mp3`.

```python
import requests

url = "https://oyqiz.airi.uz/api/v2/stt"
headers = {
    "token": "YOUR_API_TOKEN"
}
files = {
    "audio": open("audio.mp3", "rb")
}
data = {
    "language": "uz"
}

response = requests.post(url, headers=headers, files=files, data=data)
print("Транскрипция:", response.json()["transcript"])
```

## Ответ

- **transcript**: _string_  
  Текст транскрипции из аудиофайла.
- **inference_time**: _number_  
  Время, затраченное (в секундах) на обработку аудио и создание транскрипции.

### Пример Ответа (200 OK)

```json
{
  "transcript": "Haq olinur, berilmas.",
  "inference_time": 2.1
}
```

### Пример Ошибки (429 Too Many Requests)

```json
{
  "error": "Превышен лимит скорости. Разрешено 5 минут аудио в минуту.",
  "status": 429
}
```

## Примечания

- Модель STT оптимизирована для точной транскрипции, с автоматическим определением языка, если `language` не указан.
- Поддерживаемые аудиоформаты: `MP3`, `WAV`, `AAC` и `OGG` (подробности в [API Reference](../../api/stt/single)).
- Производительность может варьироваться в зависимости от качества и длины аудио.
- Оплата рассчитывается на основе длительности обработанного аудио, измеряемой в секундах.
- Проверьте [дашборд](https://voice.airi.uz) для деталей использования и поддержки.
