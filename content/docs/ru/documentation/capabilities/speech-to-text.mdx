# Преобразование речи в текст

DVoice предоставляет мощные возможности преобразования речи в текст (STT), позволяя преобразовывать устную речь в точные текстовые транскрипции. Эта функция поддерживается одной конечной точкой, разработанной для удобства использования и автоматического определения языка при необходимости.

## Справочник API

<Callout type="info">
  Для получения подробных инструкций по использованию конечной точки STT,
  включая аутентификацию, поддерживаемые аудиоформаты и обработку ошибок,
  обратитесь к [справочнику API](../../api/stt/single).
</Callout>

## Обзор конечной точки

Функциональность STT доступна через POST-запрос к конечной точке `https://oyqiz.airi.uz/api/v2/stt`. Эта конечная точка обрабатывает аудиовход и возвращает транскрибированный текст вместе с метриками производительности.

## Требования к вводу

- **Аудиовход**: Конечная точка принимает аудиофайлы, отправленные как `form-data` с именем поля `audio`. Поддерживаемые аудиоформаты будут подробно описаны в справочнике API.
- **Опциональное поле языка**: Можно указать параметр языка; если он оставлен пустым, система автоматически определяет язык аудио.

## Структура ответа

Конечная точка возвращает объект JSON с следующими полями:

- **transcript**: Строка, содержащая транскрибированный текст из аудио.
- **inference_time**: Число, представляющее время (в секундах), затраченное на обработку аудио и генерацию транскрипции.

## Примечания по использованию

- Модель STT оптимизирована для точной транскрипции, с автоматическим определением языка как запасным вариантом, если язык не указан.
- Производительность может варьироваться в зависимости от качества и длины аудио.
- **Тарификация**: Затраты рассчитываются на основе длительности обработанного аудио, измеряемой в секундах.
