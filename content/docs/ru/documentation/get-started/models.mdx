Платформа DVoice предлагает широкий спектр моделей для функций преобразования текста в речь (TTS) и речи в текст (STT). Ниже вы можете ознакомиться с доступными моделями и их предназначением.

## Модели преобразования текста в речь (TTS)

DVoice предоставляет две модели TTS, каждая из которых разработана для создания естественного звучания речи:

- **Surayyo_v2**: Усовершенствованная версия модели Surayyo с улучшенной четкостью голоса, выразительностью и поддержкой эмоциональных тонов голоса. Идеально подходит для приложений, требующих динамичного и привлекательного аудиовыхода.
- **Iroda**: Универсальная модель TTS с уникальным профилем голоса, подходящая для различных случаев использования — от повседневных до профессиональных.

## Модели преобразования речи в текст (STT)

DVoice в настоящее время поддерживает одну модель STT, оптимизированную для точной транскрипции устной речи:

- **STT модель**: Эта модель разработана для преобразования речи в текст с высокой точностью. Дополнительные сведения о ее производительности и поддерживаемых языках будут добавлены по мере ее развития.

## Сравнение моделей

| Модель     | Тип | Ключевые особенности                                      |
| ---------- | --- | --------------------------------------------------------- |
| Surayyo_v2 | TTS | Улучшенная четкость, выразительность, эмоциональный голос |
| Iroda      | TTS | Универсальный голос, широкий спектр случаев использования |
| STT        | STT | Точная транскрипция                                       |
