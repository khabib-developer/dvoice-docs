# Обзор API TTS

API DVoice TTS (Text-to-Speech) преобразует текст в высококачественное аудио. Поддерживает два endpoint'а:

- **Одноразовый Endpoint**: Генерирует аудио из текста за один HTTP POST-запрос ([Подробности](./4-single)).
- **Потоковый Endpoint**: Транслирует аудио в реальном времени через WebSocket для приложений с низкой задержкой ([Подробности](./5-stream)).

## Возможности

- Поддержка нескольких аудиоформатов (например, MP3, WAV).
- Настраиваемые модели голоса (например, `Iroda`).
- SDK для [Node.js](https://www.npmjs.com/package/dvoice-tts) и [Python](https://pypi.org/project/dvoice-tts) ([Использование SDK](./3-sdks)).

## Примеры использования

- Озвучка для видео или подкастов.
- Наррация в реальном времени для приложений.
- Инструменты доступности для слабовидящих пользователей.

## Начало работы

1. Получите API-токен ([Аутентификация](./2-authentication)).
2. Используйте наши [SDK](./3-sdks) для быстрой интеграции.
3. Изучите [Одноразовый Endpoint](./4-single) или [Потоковый Endpoint](./5-stream).
