# TTS API haqida umumiy ma'lumot

DVoice TTS (Text-to-Speech) API matnni yuqori sifatli audiolarga aylantiradi. U ikkita endpoint’ni qo'llab-quvvatlaydi:

- **Bir martalik Endpoint**: Matnni bitta HTTP POST so'rovi orqali audioga aylantiradi ([Tafsilotlar](./4-single)).
- **Oqim Endpoint**: Real vaqt rejimida audio oqimini WebSocket orqali ta'minlaydi, past kechikishli ilovalar uchun mos ([Tafsilotlar](./5-stream)).

## Xususiyatlar

- Bir nechta audio formatlarni qo'llab-quvvatlash (masalan, MP3, WAV).
- Sozlanadigan ovoz modellar (masalan, `Iroda`).
- [Node.js](https://www.npmjs.com/package/dvoice-tts) va [Python](https://pypi.org/project/dvoice-tts) uchun SDK’lar ([SDK’dan foydalanish](./3-sdks)).

## Foydalanish holatlari

- Video yoki podkastlar uchun ovoz berish.
- Ilovalar uchun real vaqt rejimida hikoya qilish.
- Ko'rish qobiliyati cheklangan foydalanuvchilar uchun maxsus imkoniyatlar vositalari.

## Boshlash

1. API tokenini oling ([Autentifikatsiya](./2-authentication)).
2. Tez integratsiya uchun [SDK’larimizdan](./3-sdks) foydalaning.
3. [Bir martalik Endpoint](./4-single) yoki [Oqim Endpoint](./5-stream) ni o'rganing.
