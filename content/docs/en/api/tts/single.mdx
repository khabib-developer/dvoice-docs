The **Single Endpoint** lets you send text and receive a complete audio file in one HTTP request.

## Endpoint

`POST https://oyqiz.airi.uz/api/v2/tts`

## Authentication

Include your API token in the request header:

```
token: <your-token>
```

## Request Body

| Field  | Type   | Required | Description                                        |
| ------ | ------ | -------- | -------------------------------------------------- |
| text   | string | ✅       | The text to synthesize (max 5000 chars).           |
| model  | string | ✅       | Voice model (`Iroda` or `Surayyo_v2`).             |
| format | string | ✅       | Output format: `MP3`, `WAV`, `AAC`, `OGG`, `FLAC`. |
| data   | object | ❌       | Extra options (only for `Surayyo_v2`).             |

### Example (JSON)

```json
{
  "text": "Hello world",
  "model": "Iroda",
  "format": "MP3"
}
```

## Example Usage

<Tabs items={['curl', 'javascript', 'python']}>
<Tab value="curl">

```bash
curl -X POST "https://oyqiz.airi.uz/api/v2/tts"   -H "Content-Type: application/json"   -H "token: YOUR_API_TOKEN"   -d '{
    "text": "Hello world from DVoice",
    "model": "Iroda",
    "format": "MP3"
  }'   --output output.mp3
```

</Tab>
<Tab value="javascript">

```javascript
import fetch from "node-fetch";
import fs from "fs";

async function tts() {
  const response = await fetch("https://oyqiz.airi.uz/api/v2/tts", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      token: "YOUR_API_TOKEN",
    },
    body: JSON.stringify({
      text: "Hello world from DVoice",
      model: "Iroda",
      format: "MP3",
    }),
  });

  const buffer = await response.arrayBuffer();
  fs.writeFileSync("output.mp3", Buffer.from(buffer));
}

tts();
```

</Tab>
<Tab value="python">

```python
import requests

url = "https://oyqiz.airi.uz/api/v2/tts"
headers = {
    "Content-Type": "application/json",
    "token": "YOUR_API_TOKEN"
}
payload = {
    "text": "Hello world from DVoice",
    "model": "Iroda",
    "format": "MP3"
}

response = requests.post(url, json=payload)
with open("output.mp3", "wb") as f:
    f.write(response.content)
```
